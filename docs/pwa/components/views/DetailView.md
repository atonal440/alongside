# pwa/src/components/views/DetailView.tsx

## Components

**`DetailView`** — Read-only detail screen for a single task. Shows all task fields: title, status badge, due date, recurrence, project, full notes, kickoff note, and the tasks it blocks / is blocked by (using link maps). Action buttons: Mark Done, Reopen, Edit, Back. Navigates to `EditView` on Edit.
