# pwa/src/components/views/SuggestView.tsx

## Components

**`SuggestView`** тАФ The default "Suggest" view. Shows a single task at a time from the `suggestQueue`, presented as a large `TaskCard`. Buttons: Start (activates the task), Skip (snoozes briefly), Mark Done, Edit. A counter below the card shows how many more tasks are in the queue. Shows `EmptyState` when the queue is empty. Includes `AddBar` for creating new tasks.
