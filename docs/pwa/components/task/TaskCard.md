# pwa/src/components/task/TaskCard.tsx

## Components

**`TaskCard`** — Full-detail task card used in `AllView` and `SessionView`. Displays title, `TaskMeta` metadata, truncated notes, and a row of action buttons (Mark Done, Skip/Snooze, Start, Edit/Detail). Calls the appropriate action creator for each button. Highlighted differently when the task is active vs. pending.
