# pwa/src/components/task/CompactCard.tsx

## Components

**`CompactCard`** — Compact single-row task card used inside `TaskStack`. Shows a checkbox, the task title, an optional status label (including "Focused" for focused tasks), and `TaskMeta` metadata (due date, recurrence). Accepts `onComplete` and `onDetail` callbacks. Used in stacked dependency views rather than the primary suggestion card.
